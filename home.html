{modest maps}

Modest Maps is a family of minimalist map APIs, targeting different platforms. The goal is to provide a minimal, extensible, customizable, and free basis for mapping.

modestmaps.js

A Javascript API for embeddable web maps. If you don't know where to start, this is most likely what you're looking for.

modestmaps-cinder

Utilities in Cinder, a C++ library for creative coding

modestmaps-as3

An Actionscript 3 library for Flash and Flex applications

modestmaps-processing

A Processing library for creative coding in the Processing language

modestmaps-of

Utilities in the OpenFrameworks library, another C++ environment for visual code
